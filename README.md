# Tab Groups Manager

A simple Chrome extension for managing tab groups using the chrome.tabGroups API, generated by Claude 3.7 Sonnet:

- https://claude.ai/chat/92b6125c-de2b-4ecc-8c3c-f7044c98191d
- https://claude.ai/chat/edaed6d1-6db4-46d5-9176-3cf5d18135e2

## Features

- View all existing tab groups organized by window
- See the number of tabs in each group
- Expand or collapse tab groups
- Close tab groups (and all tabs within them) with a confirmation dialog
- Create new tab groups from the current tab
- Choose colors for new tab groups
- Filter tab groups by name with search functionality
- Sort tab groups by various criteria (name, color, tab count, creation date, modification date)
- Debug view to see all details about a tab group in JSON format
- Automatic tracking of when groups are created and modified

## Installation Instructions

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top-right corner
4. Click "Load unpacked" and select the directory containing the extension files
5. The extension icon should appear in your Chrome toolbar

## Permissions

This extension requires the following permissions:

- `tabGroups`: To interact with Chrome's tab groups
- `tabs`: To access and manage tabs
- `windows`: To get information about Chrome windows
- `storage`: To store creation and modification dates for tab groups

## Creating Icons

You'll need to create icons for your extension in the following sizes:
- 16x16 px
- 48x48 px
- 128x128 px

Place these in an `images` folder with the filenames shown in the file structure above.

## How to Use

1. Click the extension icon in your toolbar
2. View existing tab groups organized by Chrome window
3. Use the search box to filter groups by name
4. Use the sort dropdown to organize groups by different criteria
5. Use the buttons to:
   - Expand/collapse tab groups
   - Close tab groups (with confirmation)
   - View debug information about a tab group
6. To create a new group:
   - Enter a name in the "Group Name" field
   - Select a color from the dropdown
   - Click "Create Group from Current Tab"

## Development

This is a basic proof-of-concept that demonstrates the `chrome.tabGroups` API. You can extend it with additional features such as:

- Renaming existing groups
- Changing colors of existing groups
- Moving tabs between groups
- Saving favorite group configurations
- Keyboard shortcuts for common actions
